# DATUM Gateway with hooks

A hook is added for accepted work.

## New config elements

The username of the proxypool at the upstream pool (typically a bitcoin address, "bc1xxxsomething..."):

```
	"proxypool": {
		"upstream_username": "bc1xxxsomething"
	}
```
